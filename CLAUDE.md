# CLAUDE.md

## Описание проекта

SapAutomation — фреймворк на .NET 8 для автоматизации SAP GUI (SAP GUI Scripting).

Решение (`SapAutomation.sln`) состоит из:
- **SapAutomation.Core** — ядро/бизнес-логика фреймворка.
- **SapAutomation.Tests** — модульные тесты (NUnit).

## Первое задание для Claude

Прочитать CLAUDE.md и осмотреть решение.

Сначала:
1. Запустить `dotnet build` и `dotnet test`.
2. Удалить дефолтные placeholder-файлы, если они не нужны.
3. Создать первоначальную чистую архитектуру (Clean Architecture) для фреймворка автоматизации SAP GUI.
4. Сохранить net8.0 и NUnit.
5. Пока не добавлять зависимости SAP COM.
6. Не реализовывать бизнес-специфичные SAP-сценарии.
7. Объяснить планируемую структуру перед тем, как вносить изменения в файлы.
